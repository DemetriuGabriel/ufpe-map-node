# UFPE-MAP

## História dos Usuários

```
Como estudante da CAC
Eu gostaria de poder ser informado sobre eventos de outros centros
Para poder expandir meus conhecimentos  e conhecer novas áreas e pessoas
```

```
Como assessoria de comunicação do CIN
Eu gostaria de ter outras formas de informar a comunidade UFPE sobre eventos
Para atrair mais pessoas de diferentes áreas e diversificar o público
```

```
Como organizador de eventos
Eu gostaria de ter uma forma mais simples de atingir o público UFPE
Para atingir meu público alvo em eventos acadêmicos
```

```
Como um aluno da UFPE
Eu gostaria de avaliar os eventos que estão disponiveis e poder ver a avaliação de outros usuários 
Para poder me certificar se o evento é confiável e informar a comunidade se este é um evento útil
```
